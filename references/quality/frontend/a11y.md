# A11Y — Accessibility

### A11Y-001: Interactive elements keyboard-reachable + labeled
- **Trigger:** adding interactive UI.
- **Required behavior:** focusable, operable by keyboard, with accessible names.
- **Verification signal:** axe / linter; reviewer.
- **Severity:** WARN
- **Enforcer:** linter + reviewer

### A11Y-002: Color contrast meets WCAG AA
- **Trigger:** color/typography changes.
- **Required behavior:** contrast ratio ≥ WCAG AA.
- **Verification signal:** axe.
- **Severity:** WARN
- **Enforcer:** linter

### A11Y-003: No layout break at standard breakpoints
- **Trigger:** layout changes.
- **Required behavior:** usable at common breakpoints.
- **Verification signal:** reviewer.
- **Severity:** ADVISE
- **Enforcer:** reviewer
